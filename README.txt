Compilation : make (dans le terminal) pour compiler l'intégralité du projet. 
Utilisation : ./Jeu pour lancer le jeu, possibilité de rajouter des options au lancement du jeu (-A pour les auteurs, -V pour la
version, -H pour l'aide du jeu qui est disponible aussi en jeu, -All pour les 3 options précédentes et -C + le fichier de configuration à passer 
en paramètre au programme.)
